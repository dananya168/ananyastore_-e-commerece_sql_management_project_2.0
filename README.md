# E-Commerce SQL Management Project

This project is a sample e-commerce store database created using SQL.

## Files Included
- ananyashop create tables.sql.sql
- ananyashop insert tables.sql.sql
- ananyashop queries.sql.sql

## Database Tables
- Customers
- Products
- Orders
- Order Items
- Payments

## Purpose
The project demonstrates:
- Table creation
- Data insertion
- SQL queries using JOIN, GROUP BY, SUM, COUNT, and revenue calculations

## How to Run
1. Run the create tables file
2. Run the insert tables file
3. Run the queries file
